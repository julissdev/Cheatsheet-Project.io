HTML&CSS Cheatsheet

HTML Tags

Tag Name Description
<html> HTML Element Root of an HTML document
<em> Emphasis Will emphisize words in italics
<strong> Strong Will emphisize words in bold
<p> Paragraph Allows you to create a paragraph element
<br> Break Create a line break. Does not have closing tag

HTML Attributes

Tag Name Description
id Identifier Accepts a single value and is used once per page
class Class Accepts multiple values and can be used several times per page
